F.A.Q - SUPPORT

1. Quel est le dépôt minimum ?+
Le dépôt minimum conseillé est de 10 USDT sur votre compte Spot pour permettre au robot d'exécuter les ordres.
2. Comment lier mon API ?+
Générez une clé API sur BingX ou MEXC avec les permissions "Spot Trading" activées, puis collez-les dans l'onglet AUTO.
3. Quelles cryptos sont analysées ?+
Le robot scanne 50 actifs majeurs dans les narratives IA, RWA et DePIN pour une accumulation stratégique.
4. Est-ce sécurisé ?+
Oui. Le robot ne peut pas retirer vos fonds. Il a seulement l'autorisation d'acheter et de vendre sur votre compte.
5. Pourquoi H4/H1/M30 ?+
C'est une stratégie en cascade : le robot cherche d'abord des signaux forts en 4h, puis descend en 1h et 30m si nécessaire.
6. Puis-je arrêter le robot ?+
Oui, utilisez le bouton "ARRÊT D'URGENCE" sur le Dashboard pour stopper immédiatement toutes les analyses.
7. Qu'est-ce que le CRT ?+
Le Candle Range Theory analyse les balayages (sweeps) de liquidité pour acheter au meilleur prix institutionnel.
8. Où sont mes profits ?+
Vos profits restent sur votre portefeuille Spot. Le robot accumule les jetons directement sur votre compte.
9. Support Telegram ?+
Toutes les notifications d'achat sont envoyées en temps réel sur votre canal Telegram configuré.
10. Frais de service ?+
Le robot BLACKSNIPER est un outil interne du labo Sangmelima. Aucun frais n'est prélevé sur vos trades.
